# 信用卡回饋查詢

一個查信用卡回饋的小工具，兩種查法：

- **🔍 依場景**：選「行動支付」或「日本消費」，再選通路／店家（日本可細到**逐店**，例如藥妝→大國／松本清…），告訴你該刷哪張卡回饋最高。
- **💳 依卡片**：選一張卡，看它**所有回饋場景與指定通路**（含切權益卡 國泰 CUBE／台新 Richart 的各方案）。

也可以直接用上方**搜尋**打店家／通路／卡名關鍵字（例如：藥妝、唐吉訶德、Netflix、Suica、PayPay）。
